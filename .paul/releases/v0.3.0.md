# Release Notes: v0.3.0

Released: 2026-04-20

## Summary

v0.3.0 completes operational workflow intelligence:
- git-ledger context and drift intelligence integrated into orchestration
- apply-time escalation, recovery playbooks, and high-risk mutation guardrails
- operator run-manifest and release-readiness gate contracts
- end-to-end operational acceptance coverage for release-ready and release-hold journeys

## Highlights

1. Git-ledger intelligence
- Git ledger baseline contract with repository availability and change counters
- Deterministic change classification and drift severity signals
- Ledger-aware next-action and guidance tuning

2. Apply safety and recovery
- Tiered apply escalation contract (`none`, `advisory`, `guarded`, `blocked`)
- Deterministic recovery playbook contract by escalation reason
- High-risk mutation guardrails for irreversible recommendations and critical risks

3. Operator surfaces and release ops
- Operator run-manifest contract for route/mode/specialist/artifact/safety/ledger summary
- Release-readiness gate contract with deterministic required gates and failure summaries
- Explicit release-hold signals with actionable next gate action output

4. Acceptance hardening
- End-to-end operational acceptance journeys for:
  - apply success with release-ready outcome
  - high-risk guardrail release hold
  - validation failure recovery + release hold
  - critical conflict release hold
