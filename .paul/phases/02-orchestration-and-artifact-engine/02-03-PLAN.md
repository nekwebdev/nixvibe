<objective>
## Goal
Implement the artifact output pipeline for `init` and `audit` routes, including mode-aware propose/apply behavior.

## Purpose
Complete Phase 2 by turning merged orchestration output into concrete scaffold/refactor artifacts that users can act on immediately.

## Output
A deterministic artifact engine that can emit:
- init scaffold artifacts (`flake.nix`, module defaults, architecture/next-step docs)
- audit refactor artifacts (`REFACTOR_PLAN.md`, `TARGET_TREE.md`, patch proposal metadata)
with write behavior gated by mode (`advice`, `propose`, `apply`).
</objective>
