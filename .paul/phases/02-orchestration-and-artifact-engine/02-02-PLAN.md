```gherkin
Given a bounded list of specialist tasks
When orchestration dispatch runs
Then specialists execute in parallel and return per-specialist results with stable ordering
```

## AC-2: Structured Payload Validation
```gherkin
Given specialist outputs as dictionaries
When payload validation runs
Then missing required fields or invalid confidence values are rejected and excluded from merge
```

## AC-3: Deterministic Merge and Safety Gating
```gherkin
Given multiple valid specialist payloads with conflicts
When merge resolution runs
Then conflict handling follows policy ordering and contradictory critical findings force final mode to propose
```

## AC-4: Unified Orchestration Result Contract
```gherkin
Given at least one successful specialist payload
When pipeline unification runs
Then output includes selected mode, artifact summary, and one immediate next action
```

